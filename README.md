# Dserve assignment: SnakeAndLadder

## Running programme and test

----

### Run the programme

----

```shell
dotnet build
dotnet run
```

### Run the tests

----

### Execute all tests

----

``` shell
cd SnakeLadderGameEngine.Tests
dotnet test
```

### Execute specific test

dotnet test --filter DisplayName = XUnitNamespace.TestClass1.Test1
